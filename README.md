# 📊 E-Commerce Performance Dashboard (Power BI)

## 🔍 Overview
This project presents an interactive Power BI dashboard analyzing e-commerce transactions to derive business insights.

The dashboard focuses on revenue trends, order distribution, discount impact, and payment method performance.

---

## 📈 Key Metrics

- Total Revenue: 757K+
- Total Orders: 3,660
- Average Order Value: 206.91
- Revenue Growth % (MoM): 13.33%

---

## 🧠 Business Insights

- Identified top revenue-generating categories
- Analyzed discount impact across categories
- Evaluated payment method revenue contribution
- Implemented Month-over-Month revenue growth using DAX

---

## 🛠 Tools Used

- Power BI
- DAX (DATEADD, CALCULATE, DIVIDE)
- Data Modeling
- Dashboard Design (Dark Executive Theme)
